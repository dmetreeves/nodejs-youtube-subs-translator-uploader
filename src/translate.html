Перевести Загрузить Все языки

Автоматический переводчик субтитров по всем языкам на yandex translate

После нажатия кнопки "Перевести" на сервере начнется процесс перевода текста субтитров из поля "Субтитры" по всем языкам из поля "Языки". Это может длиться до 5 минут. Затем автоматически скачается zip-файл с названием из поля "Файл" со всеми переводами внутри.
